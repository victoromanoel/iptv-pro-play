# 📺 IPTV Pro Player - APK

Player IPTV com canais 24h integrados.

## 🚀 Como gerar o APK

1. Faça o upload deste projeto no GitHub
2. Vá em **Actions** → o build iniciará automaticamente
3. Aguarde ~5 minutos
4. Baixe o APK em **Actions → seu build → Artifacts**

> ✅ O APK também é gerado automaticamente em cada push na branch `main`.
